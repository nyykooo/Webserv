HTTP METHODS

HTTP defines special words called methods to describe what the client wants:

GET:
The client sends a request line like: 


						function helloWorld() {
							console.log("Hello, world!");
						}

POST: Used to send data to the server.
Example: submitting a form, uploading a file, creating a new record in a database.
Not idempotent (repeating may cause duplicates).

DELETE: Used to remove data on the server.
Example: deleting a user account or removing a file.
Idempotent (deleting the same resource twice still results in "gone").
